# Fact-Check Report — 2026-05-17-002-apob-95-dentro-da-proto

**Verdict: REVISE**

Draft é tecnicamente sólido e o frame editorial (faixa de laboratório ≠ faixa de risco ótimo) é defensável e bem ancorado em literatura cardiovascular preventiva. Dois ajustes técnicos: (1) o threshold <60 mg/dL diverge do consenso ESC/EAS (<65 para very-high risk) — corrigir para preservar credibilidade; (2) 'vai infartar' precisa suavização para evitar determinismo. Recomenda-se também atribuição mínima (ESC/EAS 2019) já que o post se posiciona como rigor técnico contra referência laboratorial.

## Claims auditados


### 🟡 CITE-SOURCE · evidência 4/5
> "A medicina cardiovascular preventiva trabalha com outro número: ApoB abaixo de 80 mg/dL para risco padrão"

- **Tipo**: guideline
- **Reasoning**: Threshold de ApoB <80 mg/dL é consistente com guidelines ESC/EAS e literatura preventiva, mas o número exato não é universal — alguns consensos usam <90 para risco moderado e <80 para alto risco. Precisa atribuição.
- **Fontes encontradas**: ESC/EAS 2019 Dyslipidaemia Guidelines (Mach F. et al., Eur Heart J 2020); AHA/ACC 2018 Cholesterol Guideline
- **Sugestão reescrita**: 
  > A medicina cardiovascular preventiva (ESC/EAS 2019) trabalha com alvos mais agressivos: ApoB abaixo de ~80 mg/dL para risco moderado.


### 🟡 SOFTEN · evidência 4/5
> "abaixo de 60 para quem já tem fator de risco"

- **Tipo**: guideline
- **Reasoning**: Guideline ESC/EAS define ApoB <65 mg/dL para very-high risk, não <60. O número está próximo mas não é exato — risco de imprecisão técnica num post que se vende como rigor técnico.
- **Fontes encontradas**: ESC/EAS 2019 Dyslipidaemia Guidelines — ApoB <65 mg/dL para very-high risk, <80 para high risk
- **Sugestão reescrita**: 
  > abaixo de 65 mg/dL para quem já tem fator de risco cardiovascular estabelecido (ESC/EAS 2019).


### 🟢 KEEP · evidência 4/5
> "a referência do laboratório é estatística — pega a média da população"

- **Tipo**: mechanism
- **Reasoning**: Afirmação metodologicamente correta: intervalos de referência são derivados de percentis (2.5-97.5) de população amostral, não de outcome clínico.
- **Fontes encontradas**: Princípio padrão de definição de intervalos de referência laboratorial (IFCC/CLSI EP28-A3c)



### 🟡 SOFTEN · evidência 4/5
> "É o limite alto de uma média que inclui muita gente que vai infartar"

- **Tipo**: causal
- **Reasoning**: Claim é direcionalmente correto (ApoB elevado dentro da 'faixa normal' associa-se a risco residual significativo) mas a frase 'vai infartar' é determinística demais — beira fear-mongering. Suavizar para risco aumentado, não desfecho garantido.
- **Fontes encontradas**: Sniderman AD et al., JAMA Cardiol 2019 — ApoB como preditor superior de eventos CV; Mendelian randomization studies (Ference BA et al., JAMA 2019)
- **Sugestão reescrita**: 
  > É o limite alto de uma média que inclui muita gente com risco cardiovascular residual elevado.
