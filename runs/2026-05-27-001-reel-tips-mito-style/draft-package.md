# Draft — Reel 8 dicas Mito-style

## Caption

8 dicas. Cada uma calibrada num biomarcador específico do painel.

Não é checklist genérico — é o que move os números quando lidos em conjunto.

→ Glicose: carbo com proteína + fibra para a curva.
→ Sono: janela 60min travada — o circadiano não negocia.
→ Ferritina: heme + vitamina C juntos, longe de café.
→ Massa magra: -10%/década sem força. Comece hoje.
→ Movimento: hs-CRP responde antes da dor.
→ ApoB: a partícula que entope, não o número que tranquiliza.
→ Respiração: nariz acende o vagal. Boca acende o cortisol.
→ Tela 90min antes do sono: cortisol e tireoide reorganizam devagar.

Cada uma vira número no painel. Cada número vira protocolo.
